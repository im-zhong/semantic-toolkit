API调用演示

POST /api/analyze

请求参数

参数名类型必填说明
abstractstring是英文摘要文本

语步标签

BACKGROUND - 研究背景 PURPOSE - 研究目的 METHOD - 研究方法 RESULT - 研究结果 CONCLUSION - 研究结论

Python示例

import requests

url = "http://localhost:5001/api/analyze"
data = {"abstract": "Your abstract text"}
response = requests.post(url, json=data)
print(response.json())

curl示例

curl -X POST http://localhost:5001/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"abstract": "Your abstract text"}'
