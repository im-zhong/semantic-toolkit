API调用演示

POST /api/analyze

请求参数

参数名类型必填说明
textstring是基金项目文本

语步标签

basis - 立项依据 objective - 研究目标 content - 研究内容 approach - 技术路线 expected_result - 预期成果 application_value - 应用价值

Python示例

import requests

url = "http://localhost:5002/api/analyze"
data = {"text": "您的项目文本"}
response = requests.post(url, json=data)
print(response.json())

curl示例

curl -X POST http://localhost:5002/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"text": "您的项目文本"}'

返回示例

{
    "sentences": [
        {"句子内容": "...", "语步标签": "basis"},
        {"句子内容": "...", "语步标签": "objective"}
    ]
}
