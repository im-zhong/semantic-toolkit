Python 调用示例
import requests

url = "http://localhost:7862/api/move-recognition/english-abstract"
headers = { "Content-Type": "application/json" }
payload = { "text": "An English scientific abstract text here" }
resp = requests.post(url, json=payload, headers=headers)
print(resp.json())
接入建议

1. 业务系统仅需调用统一业务接口，无需关注底层模型选型与提示词细节。

2. 返回结构固定为 background / purpose / method / result / conclusion，适合前端展示与结构化存储。

3. 建议通过服务端完成鉴权与日志记录，便于后续接口治理与能力审计。
