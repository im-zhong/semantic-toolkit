# 科技文献语步识别工具

从科技文献摘要中自动识别并标注研究背景、研究目的、研究方法、研究结果、研究结论。

## 功能特点

- 🔍 **语步识别**：自动识别5种学术语步
- 🌐 **Web界面**：简洁易用的Gradio界面
- ⚡ **API调用**：无需本地算力，通过智谱AI API实现
- 📝 **提示词工程**：精心设计的提示词模板，确保识别准确性

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 获取API密钥

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册并登录
3. 在控制台获取API密钥

### 3. 运行程序

```bash
python app.py
```

然后在浏览器中打开 `http://127.0.0.1:7860`

### 4. 使用步骤

1. 输入API密钥并点击"连接API"
2. 在输入框中粘贴科技文献摘要
3. 选择模型（推荐 GLM-4-Flash）
4. 点击"开始分析"

## 语步类型说明

| 语步类型 | 英文 | 说明 | 典型关键词 |
|---------|------|------|-----------|
| 研究背景 | Background | 介绍研究领域现状、问题、重要性 | 目前、现有、然而、存在问题 |
| 研究目的 | Purpose | 说明研究目标、创新点 | 本文、旨在、提出、设计 |
| 研究方法 | Method | 描述方法、技术、实验设计 | 采用、使用、方法、模型、算法 |
| 研究结果 | Result | 展示发现、性能指标 | 结果表明、发现、提升、准确率 |
| 研究结论 | Conclusion | 总结贡献、意义、展望 | 结论、验证、意义、贡献、未来 |

## 模型选择建议

| 模型 | 特点 | 适用场景 |
|-----|------|---------|
| GLM-4-Flash | 快速、免费额度多 | 日常使用（推荐） |
| GLM-4 | 标准性能 | 需要更好效果时 |
| GLM-4-Plus | 最强性能 | 复杂分析任务 |

## 扩展开发

### 自定义提示词

修改 `prompts.py` 中的 `SYSTEM_PROMPT` 和 `USER_PROMPT_TEMPLATE` 来自定义识别规则。

### 切换其他API

`api_client.py` 可扩展支持其他大模型API（如OpenAI、文心一言等）。
